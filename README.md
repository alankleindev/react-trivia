# React Trivia!

React Trivia is, as its name implies, a trivia game, where the Player must answer JavaScript/React trivia.

## Getting Started

Clone this repository and run npm install, then npm start.

### Prerequisites

Web Browser

## Built With

- JavaScript
- React
- Redux
